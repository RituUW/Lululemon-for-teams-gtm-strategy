# Lululemon For Teams — Go-to-Market Strategy

A product strategy case study for launching a native group-ordering platform on lululemon.com, built for running clubs and fitness communities.

## Overview

Organisers of running clubs and fitness communities already attempt group apparel orders through workarounds — WhatsApp threads, Venmo, personal accounts — proving the demand exists even though no good product serves it. Lululemon For Teams closes that gap with a native, end-to-end group-ordering experience: organisers set up an order once, members self-serve checkout and pay individually, and bulk pricing unlocks automatically.

The core bet: this isn't about creating new demand — it's about converting existing brand loyalty into community-level purchasing, capturing revenue currently lost to workarounds and competitors like Custom Ink.
